# appNest

## Description
appNest is a modern React-based web application designed for building dynamic, responsive, and interactive user interfaces. It leverages powerful tools for state management, routing, and data visualization, providing a smooth developer and user experience. The project integrates TailwindCSS for styling, React Router for navigation, and TanStack Query for efficient data fetching and caching.

## Technologies Used
### Core
- React 18
- React DOM
- React Router v7
- TanStack React Query

### Styling
- TailwindCSS
- DaisyUI
- @tailwindcss/vite

### UI & Icons
- React Icons
- Lucide React

### Charts & Visualization
- Recharts

### Notifications
- Notiflix

### Development & Linting
- Vite
- ESLint
- @eslint/js
- eslint-plugin-react-hooks
- eslint-plugin-react-refresh
- TypeScript types for React (`@types/react`, `@types/react-dom`)
